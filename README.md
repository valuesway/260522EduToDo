# 🎓 에듀투두 (EduTodo) - 교육자를 위한 스마트 교무 수첩

> **바쁜 교사, 강사, 교수 등 교육자분들의 하루를 획기적으로 돕는 맞춤형 스마트 할 일 관리(To-Do) 및 대시보드 플래너입니다.**
> 
> 복잡한 가입이나 서버 설정 없이 브라우저에서 즉시 작동하며, Netlify에 드래그 앤 드롭으로 10초 만에 전 세계에 무료 배포할 수 있도록 초경량 및 최적화되어 설계되었습니다.

---

## 🎨 주요 시각적 특징 (Design Aesthetics)

*   **프리미엄 글래스모피즘 디자인**: 투명 아크릴 질감의 디자인 시스템(`backdrop-filter: blur(12px)`)과 조화로운 섀도우를 채택하여 세련되고 모던한 작업 환경을 선사합니다.
*   **다크 & 라이트 모드 테마**: 밤샘 수업 준비나 성적 입력 시 교육자분들의 눈을 보호해 줄 다크 모드를 완벽히 지원하며, 클릭 한 번으로 손쉽게 테마를 전환할 수 있습니다.
*   **완벽한 반응형 지원**: 교무실 데스크톱의 와이드 모니터부터 이동 중 모바일 스마트폰, 태블릿(iPad 등)까지 깨짐 없이 최적의 레이아웃을 보장합니다.
*   **달성 축하 파티클(콘페티) 효과**: 할 일을 완료 처리할 때마다 화면에 터지는 화려하고 귀여운 색종이 축하 피드백 애니메이션(HTML5 Canvas 자체 엔진)이 탑재되어 성취감과 잔재미를 제공합니다.

---

## 🛠️ 교육자 특화 핵심 기능 (Core Features)

1.  **학급 및 과목별 스마트 태그 칩스**
    *   '수업준비', '행정업무', '학생상담' 등 고유 분류뿐만 아니라, **교과명이나 담당 학급('3학년 1반', '수학 A반' 등)을 언제든 추가 및 삭제**하여 필터링해 볼 수 있습니다.
2.  **교무 특화 원클릭 퀵 템플릿**
    *   "지필평가 시험 출제", "나이스 생기부 입력", "수업 PPT 및 인쇄물 제작", "주간학습안내 등록", "공문 결재 상신" 등 가장 자주 일어나는 교사들의 업무를 클릭 한 번으로 자동 입력할 수 있습니다.
3.  **오늘의 수업 시간표 위젯**
    *   사이드바에 1교시부터 6교시까지의 시간표와 함께 핵심 준비물이나 메모를 밀착 기록할 수 있습니다. 모달창을 통해 즉각적인 수정과 LocalStorage 저장이 이루어집니다.
4.  **학생 상담 & 관찰 일지 전용 메모장**
    *   수업 태도 관찰, 상담 이력 기록, 학부모 유선 면담 사항 등을 학생 이름별 카드로 누적 보관 및 관리하고, 실시간 키워드 검색을 통해 빠르게 탐색할 수 있습니다.
5.  **무설정 로컬 저장 및 데이터 백업(Export/Import JSON)**
    *   모든 데이터는 별도의 데이터베이스 없이 사용자의 웹 브라우저(`localStorage`)에 자동으로 영구 보존됩니다.
    *   기기를 바꾸거나 브라우저 캐시 초기화에 대비해 전체 데이터를 단 한 번의 클릭으로 `.json` 파일로 내보내어 소중한 정보를 완벽히 지키고 이식할 수 있습니다.

## 📂 프로젝트 파일 구조 (File Structure)

```text
├── index.html       # 시맨틱 마크업, 구글 폰트, Feather 아이콘 로드 및 레이아웃 정의
├── style.css        # CSS 변수(Variables) 기반 라이트/다크 테마 및 전체 반응형 스타일링
├── app.js           # 상태(State) 관리, 시간표/학생상담 CRUD, Canvas 파티클 이펙트, 백업 로직
└── netlify.toml     # Netlify 배포 최적화, 정적 캐싱 규칙 및 강력한 보안 HTTP 헤더 탑재
```

## 🌐 Netlify 배포 방법 (How to Deploy)

**에듀투두**는 빌드 도구나 설정이 전혀 필요 없는 정적 싱글 페이지 웹앱입니다. 다음 2가지 방법 중 하나로 단 1분 만에 전 세계에 무료 배포할 수 있습니다.

### 방법 1: Netlify Drop (가장 추천!)
1. 웹 브라우저에서 [Netlify Drop](https://app.netlify.com/drop) 페이지로 이동합니다.
2. 본 프로젝트의 전체 폴더(파일 4개가 들어 있는 폴더)를 마우스로 선택해 업로드 창에 끌어다 넣습니다.
3. 3초 안에 고유한 도메인 주소(예: `https://edu-todo.netlify.app`)와 함께 배포가 끝납니다!

### 방법 2: GitHub 연결 배포
1. 깃허브 저장소(`https://github.com/valuesway/260522EduToDo`)에 본 소스코드를 업로드합니다.
2. [Netlify](https://netlify.com)에 로그인 후 `Add new site` -> `Import an existing project`를 눌러 해당 저장소를 연결합니다.
3. 리포지토리에 커밋이 푸시될 때마다 Netlify가 5초 이내에 새 버전을 실시간 자동 업데이트 배포합니다. (설정 내 `netlify.toml`이 자동으로 로드됩니다.)

---

## ⚙️ 로컬 실행 방법 (Local Run)

인터넷 연결이 원활하지 않은 오프라인 환경에서도 작동합니다.
1. 본 프로젝트의 폴더를 다운로드합니다.
2. `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)로 더블클릭해서 바로 실행하시면 끝납니다!
